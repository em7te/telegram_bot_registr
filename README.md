## telegram_bot_registr

**python version:** *3.9*   

**venv:**   
*python3 -m venv venv*  
*source venv/bin/activate*

**install:**    
*pip install -r requirements.txt*

**start:**  
*добавить переменную API_TOKEN(токен бота) в bot.py, поместить в самое начало*
*python3 db/create_sql.py*  
*python3 bot.py*
